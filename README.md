# tuffsh
`tuffsh` is a GO based cross platform ssh client.
current version 1.0

### Tested Platforms
- MacOS
- Red Hat
- Windows 11

### Usage

```
 tuffsh -h       
usage: tuffsh
        [-i/--identity identity_file]
        [-k/--known-hosts known_hosts_file]
        [-p/--port ssh port number]
        [-v/--version]
        [-h/--help]
        [user@]destination[:port]
        [command]
```
